Sobre el Proyecto

Desarrollado por Pablo Conde Mellado

Este proyecto esta basado en el consumo de la api PokeApi en donde extraeremos los datos de los pókemon.

Feedback sobre la Solución

Pros:

  • Una vez acostumbrado es muy fácil de crear páginas.
  • Conexión entre el servicio y los modelos.
  • El código está bien estructurado y es fácilmente mantenible.

Contras:

  • Al principio el framework abruma y no sabes por donde empezar.
  • El tener que poner las rutas a mano, teniendo que estar muy atento.
  • Poco intuitivo para los nuevos usuarios del framework.

Elementos que más costaron:

Al principio el entender como funcionaba el framework y como utilizarlo fue de las cosas que más me costaron, junto al hecho de consumir la api y saber a que ruta acceder para obtener los datos que quería.

Elementos que menos costaron:

Creacion de los hmtl y css.
